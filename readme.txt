Streaming radio widget. 5,000 stations in more than 20 genres.

== Description ==
The 2RHYTHMS radio widget gives visitors access to 5,000 stations from your website. Choose specific genres of music or roll with them all! Works with smart / mobile devices.
= What's new in verson 1.0 of this plugin? =
1. Everything! We'll add features as time goes by...

== Installation ==
1. Download and unzip radio_widget.zip 
1. Upload the FOLDER containing `radio_widget.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. To add the radio widget on a sidebar, browse to `Appearance > Widgets` and add '2RHYTHMS Radio' to desired sidebar. Configure the parameters such as Title, Genres, and save your changes.
1. To add the radio widget to any post or page you will probably need another plugin such as the 'Widgets in pages' plugin. Direct support for this will be coming in a later release.
